Інструкція до парсеру.

1. Налаштування.
	1.1 Всі налаштування (адреса сайта, що парситься; імена вихідних файлів; назва папки з вихідними даними; більшість параметрів відбору ключових елементів (блок цитати, блок автора і т.ін)) знаходяться в файлі 'config.py'
2. Використання.
	2.1 При запуску парсера, спочатку завантажується попередньо збережена база даних (за необхідності, можна загрузити тільки деякі з гілок, наприклад, тільки Теги, або Авторів і Цитати)
	2.2 Відбувається парсинг сайту.
	Спочатку парсяться цитати. В процесі цього парсингу наповнюються дані щодо авторів (ім'я, URL та ID, що генерується під час додавання автора в базу) та дані про теги (ті ж самі поля, що і у автора).
	Парсинг тегів і авторів відбувається на основі даних, що отримані з парсингу цитат.
	Для авторів - це парсинг однієї сторінки з даними про автора.
	Для тегів - парсинг сторінки відомості про тег.
	2.3 Після парсингу відбувається експорт отриманих даних в формат JSON і в таблицю XLS
	Результатом експорту в JSON є об'єкт типу dict наступної структури:
	data = {
	"quotes": :list: [
		{"_id": :int: ID цитати,
		 "text": :str: текст цитати,
		 "author_id": :int: ID автора,
		 "author_title": :str: ім'я автора,
		 "author_url": :str: URL сторінки автора,
		 "tags": :list: [
		 	{"_id": :int: ID тега,
			 "name": :str: ім'я тега,
			 "url": :str: URL сторінки тега,
		 	}]
		}],
	"authors": :list: [
		{"_id": :int: ID автора,
		 "author_title": :str: ім'я автора,
		 "born_date": :str: дата народження,
		 "born_place": :str: місце народження,
		 "about": :str: довідка про автора,
		 "author_url": :str: URL сторінки автора,
		}],
	"tags": :list: [
		{"_id": :int: ID тега,
		 "tag_name": :str: ім'я тега,
		 "url": :str: URL сторінки тега,
		 "quotes": :list: [
		 	{"text": :str: текст цитати,
			 "author_id": :int: ID автора,
			 "author_title": :str: ім'я автора,
			 "author_url": :str: URL сторінки автора,
		 	}]
		}]
	}

3. Існує можливість отримання автора по його внутрішньому ID або по його імені, наприклад:
	get_author(data, _id=1)
    get_author(data, name="Marilyn Monroe")
Формат даних, що повертаються - JSON
