Інформація про користування API

Отримання записів по категоріях:

http://{ website-root }/api/{ category }/

Отримання запису по його ID (використовується ID з сайту Hacker News):

http://{ website-root }/api/{ ID }/

або

http://{ website-root }/api/{ category }/{ ID }/

Отримання можливих категорій для парсингу:

http://{ website-root }/api/parse

Запуск парсера по категорії:

http://{ website-root }/api/parse/{ category }/?email={ email }

Визначення термінів:

website-root
Назва сайту
category
Категорія або тип статті. Підтримуються як категорії, що зумовлені процесом отримання записів з Hacker News (jobstories, showstories, newstories і т.д.) так і категорії (типи) записів, які отримуються (job, ask, story і т.д.)
ID
ID запису з сайту Hacker News
email
email-адреса для отримання звіту про парсинг
