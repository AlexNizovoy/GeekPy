з допомогою Django REST framework створити API для наступного функціоналу:

1. Отримання списку сущностей по запросу get по категоріях some_url/category_type -  (Ask, Show, Job)
2. Отримання сущності по ID get request, наприклад:
some_url/job/<int>id
3. Запуск парсера по апі наприклад:
some_url/execute/category_name
4. По відпрацюванню парсера повинен відправлятися звіт на емейл вказаний тут
http://joxi.ru/krDMYaEI0Y57pm
5. В листі повинна міститись інформація про зроблену роботу парсером а саме, кількість зібраних новин по категоріям, затрачений час.
6. Текст емейла повинен бути у форматі НТМL, статистика у формі таблиці.
6. Завантажити результат на heroku
